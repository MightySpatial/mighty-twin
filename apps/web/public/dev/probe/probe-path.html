Click + key map

Movement continuous

  1. W · advance along centerline (t↑)
  2. S · retreat (t↓)
  3. A · yaw left
  4. D · yaw right
  5. ↑↓ · pitch (clamped ±70°)
  6. QE · roll (springs back to local up unless held)
  7. 1–5 · speed gear (caps near walls)
  8. Esc · exit Probe → fly to surface entry

HUD readout

  1. t = 0.42 · normalised position along the polyline (0 = start, 1 = end)
  2. v = 1.4 m/s · current speed in the constrained frame
  3. d⊥ = 0.18 m · orthogonal distance from centerline (clamped to radius − ε)

Auto behaviours

  1. Camera roll springs to local up after 240 ms idle (Q/E hold overrides)
  2. Vignette opacity proportional to (d⊥ - 0.6·r) / (0.4·r), capped at 0.65
  3. If a NavigableConnection is within 2·r, junction arrows appear (see junction state)
  4. If interior_tileset_url set, 3D tileset streams in; falls back to synthetic radius cylinder on load failure

Exits

  1. ESC → fly to surface, fade-to-map (1.2 s)
  2. Brand button → modal "Exit Probe?" → confirm
  3. Mode change (Atlas, Settings) → auto-saves URL t-parameter, exits cleanly
  4. Bottom-nav tap → ditto
