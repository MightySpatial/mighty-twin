Active tool

The currently selected tool is highlighted in teal across the tool grid; the status bar at the bottom mirrors the same teal accent so the user always sees which tool will fire on the next globe click.

Block level resolution

The dropdown list maps power-of-two physical sizes to internal level indices (L0–L7). Click a row to switch resolution. The engine rebuilds the active block grid at the new level — finer levels increase detail but cap the addressable volume.

Material palette

Each tile renders the chip with a darker bottom band so the user sees the side-face shading the engine will apply. The IFC class beneath drives the export and the redline/promotion pipeline (e.g. Water → IfcVoidingFeature carves the host volume, never adds mass).

Render mode

Three-way segmented control. Solid is the default; Textured pulls per-material atlases; Ray-trace switches the canvas to the progressive sampler with sun + sky controls. See the Render modes page for what each mode reveals.

Group ordering

Draw and Redline keep their existing positions. Voxel slots in third with a small NEW badge for this release.
