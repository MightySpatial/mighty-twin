Per-face overrides

Clicking a face tile opens the system file picker (or library browser if textures live in a project asset bundle). Faces with no override fall back to the material's procedural shading. Click a tile in the grid to simulate an upload.

Edge cases

Adjacent blocks of the same material seam-blend at L3 and coarser. At L2 (50 cm) and finer, every face renders as its own tile so seams remain crisp.

Per-face overrides only apply to the selected blocks. To override a whole material globally, use the material entry's edit panel.
